# 变更日志

> 所有值得注意的变更应记录在此文件中。遵循 [Keep a Changelog](https://keepachangelog.com/) 格式。

格式说明：
- `Added`：新增功能
- `Changed`：现有功能的变更
- `Deprecated`：即将移除的功能
- `Removed`：已移除的功能
- `Fixed`：Bug 修复
- `Security`：安全相关修复
